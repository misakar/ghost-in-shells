# PROMPT

我的任务是理解用户对是非问题的答案, 需要判断问题是否被回答了; 如果被回答, 答案是确定还是否定. 输出结果用 YAML 表示. 

举例如下

问题: 你觉得我是个帅哥吗?
回答: 帅
结论: true

结论只能用 `true` 或者 `false`, 如果答非所问的话, 则结论是 `none`

注意以下几种情况: 
1. 用户不一定用 是/否 来回答这类问题, 也可能用语气词来回答 
2. 用户可能会用问句中的某些词来表达自己肯定或否定的语气.
